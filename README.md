# Chat Sockets Java

Proyecto de chat en tiempo real inspirado en WhatsApp y Discord.

## Objetivo

Desarrollar una aplicación web que permita la comunicación en tiempo real entre usuarios utilizando WebSockets, persistencia en base de datos y autenticación de usuarios.

## Tecnologías

* Java
* Spring Boot
* React
* SQL
* WebSockets
* Git

## Estado del proyecto

🚧 En desarrollo

Actualmente se encuentra en fase de diseño y modelado de base de datos.

## Funcionalidades MVP

* Registro de usuarios
* Inicio de sesión
* Chat global
* Envío de mensajes
* Historial de mensajes
* Persistencia en base de datos

## Funcionalidades futuras

### Versión 1

### Versión 3
